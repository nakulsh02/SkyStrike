# SkyStrike

SkyStrike is a **modern, fast-paced web game** built to deliver smooth visuals and responsive gameplay directly in the browser. The project focuses on performance, clean design, and an engaging user experience.

## Features

- Fast and interactive gameplay
- Smooth animations and visual effects
- Clean, modern user interface
- Fully responsive (works on mobile and desktop)
- Runs directly in the browser with no setup
